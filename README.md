# 🚀 joerosslab

> fast file processor for everyday workflows

## ✨ Features

- Fast compilation (sub-second builds)
- Hot reload included
- Plugin system
- Zero config start

## 📦 Installation

```bash
npm install -g joerosslab
joerosslab init my-project
```

## 🛠️ Stack

Built with tracker + react-poll-widget
